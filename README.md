🎵Spotify Music Data Analysis
An introverts-level Data Science project that will examine the trend in music based on the functions of the Spotify audio stream to decide why songs become popular, the difference between genres and categorization of songs with the use of Machine Learning.

📊 Project Overview
The present project works with a Spotify tracks dataset to provide answers to the following three questions:

What does make songs popular features?
Are audio patterns in various genres different?
Is it possible to classify songs based on their sound through Machine Learning?
